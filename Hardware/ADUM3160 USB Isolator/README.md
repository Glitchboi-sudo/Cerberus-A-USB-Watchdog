            # Aislador USB ADUM3160

## ¿Qué es?

Este es un módulo aislador USB basado en el chip **ADUM3160**. Su función principal es proporcionar aislamiento galvánico entre tu computadora y el dispositivo USB conectado, protegiendo tu equipo contra:

- Picos de voltaje
- Ruido eléctrico
- Diferencias de potencial de tierra
- Daños por cortocircuitos

## Cómo abrir el diseño

Este proyecto está diseñado en **EasyEDA**. Para abrir y editar los archivos:

1. Abre EasyEDA (versión web o de escritorio)
2. Ve al menú superior: **Archivo → Abrir → EasyEDA...**
3. Selecciona el archivo `.json` incluido en esta carpeta
4. El diseño se abrirá en el editor
5. (Opcional) Guárdalo en un proyecto para futuras ediciones

## Notas

- Este módulo puede usarse como complemento del proyecto Cerberus para añadir una capa extra de protección
- Velocidad soportada: USB 2.0 Full Speed (12 Mbps)
